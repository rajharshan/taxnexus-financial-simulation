# TaxNexus Financial Simulation System

TaxNexus is a FinTech simulation platform that analyzes the impact of government tax increases on household financial stability.

## Features
• Disposable Income Calculation  
• Tax Impact Comparison  
• Savings Reduction Analysis  
• Budget Stress Index  
• Financial Stability Score  
• Financial Risk Classification  
• Budget Recommendations  
• Interactive Charts Dashboard  
• Downloadable Financial Report (PDF)

## Technology Stack
React.js  
Chart.js  
JavaScript  
Node.js  
jsPDF

## Installation

Clone the repository:

git clone https://github.com/rajharshan/taxnexus-financial-simulation.git

Go to the project folder:

cd taxnexus-financial-simulation/frontend/taxnexus-ui

Install dependencies:

npm install

Run the application:

npm start
